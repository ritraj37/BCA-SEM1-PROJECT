# 🧮 Simple Calculator

A basic web-based calculator that performs simple arithmetic operations such as addition, subtraction, multiplication, and division.  
This project is built using **HTML, CSS, and JavaScript** and demonstrates fundamental front-end development concepts.

---

## 🚀 Features
- Perform basic arithmetic operations (+, −, ×, ÷)
- Clean and simple user interface
- Responsive design
- Instant calculation results
- User-friendly layout

---

## 🛠️ Technologies Used
- **HTML5** – Structure of the calculator  
- **CSS3** – Styling and layout  
- **JavaScript** – Logic for calculations  

---

## 📂 Project Structure

│
├── index.html
├── style.css
├── script.js
└── README.md

## ▶️ How to Run the Project
1. Download or clone the repository
2. Open the project folder
3. Run **index.html** in your browser
